# 🛍️ Zema Boutique - Telegram Mini App

این یک پروژه فروشگاهی داینامیک و چندزبانه برای **بوتیک زِما** است که به صورت یک مینی‌اپ تلگرام طراحی شده است.

## 🚀 ویژگی‌های کلیدی (Features)
- **داینامیک بودن:** محصولات به صورت خودکار از طریق جاوااسکریپت رندر می‌شوند.
- **چندزبانه (Multi-language):** پشتیبانی کامل از زبان‌های فارسی (RTL)، انگلیسی و آلمانی.
- **سفارش آسان:** اتصال مستقیم به تلگرام برای ثبت سفارش محصول.
- **طراحی واکنش‌گرا (Responsive):** بهینه‌شده برای موبایل و دسکتاپ.

## 🛠️ تکنولوژی‌های استفاده شده (Tech Stack)
- HTML5
- CSS3 (Custom Grid & Flexbox)
- JavaScript (Vanilla JS)
- Telegram WebApp API

## 📸 نحوه نام‌گذاری تصاویر
برای اضافه کردن محصولات جدید، تصاویر باید در پوشه اصلی و با فرمت زیر ذخیره شوند:
`fotos01.jpg`, `fotos02.jpg`, ...

## 👨‍💻 توسعه‌دهنده
توسعه یافته توسط **رضا** به عنوان تمرین برنامه‌نویسی وب و مینی‌اپ‌های تلگرام.
